Tarot House Privacy Policy

Last updated: July 11, 2026

Tarot House is a Chrome extension that provides a compact floating three-card workspace for tarot readers. Tarot House does not collect, sell, transmit, or share personal information or browsing data.

1. Information We Collect

Tarot House does not collect personal information, browsing history, website content, account information, financial information, authentication information, location data, or communications.

Tarot House does not use analytics, advertising trackers, cookies, or external tracking services.

2. Local Extension Settings

Tarot House stores a limited number of interface preferences locally through the Chrome Storage API. These preferences may include:

  • The selected theme
  • The selected window size
  • The floating panel position
  • The minimized-card position
  • Whether card keywords are displayed

These settings remain in the user's Chrome profile and are not sent to the developer or to any external server.

3. Active Tab Access

Tarot House uses temporary access to the active browser tab only after the user explicitly clicks the Tarot House toolbar icon.

This access is used solely to display the Tarot House floating interface over the current webpage. Tarot House does not collect, store, analyze, or transmit the contents of the webpage.

4. Card Images and Extension Resources

Tarot card images, card names, card keywords, interface files, and visual assets are packaged locally with the extension. Tarot House does not download executable code or card data from external servers.

5. Clipboard Use

When the user selects Copy cards, Tarot House writes only the names of the currently revealed tarot cards to the clipboard. This action occurs only after an explicit user request.

Tarot House does not read existing clipboard contents.

6. Data Sharing

Tarot House does not sell, rent, disclose, or transfer user data to third parties because no user data is collected.

7. Data Retention and Removal

Tarot House does not maintain an external database of user information.

Locally stored extension settings can be removed by uninstalling Tarot House or clearing the extension's stored data through Chrome.

8. Chrome Web Store User Data Policy

Tarot House's use of information received from Chrome APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.

9. Changes to This Policy

This Privacy Policy may be updated when Tarot House features or data practices change. The latest version will always be published on this page with an updated revision date.
